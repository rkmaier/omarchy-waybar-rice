## My custom waybar rice for omarchy

[![Alt text](./images/waybar.png)](https://github.com/rkmaier/omarchy-waybar-rice)

# Waybar modules included:
- bluetooth audio battery
- btc-eruo exchange rate
- news ticker from rss links
- weather
- currency exchange
- gpu load
- network upload / download speed
- system stats

# Installation
(*Make a backup of your waybar config*)

``
git clone https://github.com/rkmaier/omarchy-waybar-rice
``

``
cp -f ~/.config/waybar
``

Copy / Overwrite the files to your ~/.config/waybar folder 
Adjust the include module script to your need
